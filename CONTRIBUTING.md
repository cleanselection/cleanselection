# Contributing

Thank you for helping improve Clean Selection.

1. Search existing issues before opening a new one.
2. Create a focused branch and keep unrelated changes separate.
3. Treat `src/cleanselection.js` as canonical; do not hand-edit generated `dist` files or WordPress core copies.
4. Run `npm run build`, `npm run minify`, `npm run docs`, `npm test`, and `npm run check`. Fetch the pinned minifier first with `npm run fetch:minifier` when it is not cached.
5. Update documentation and types when behavior or public contracts change.
6. Submit a pull request explaining the problem, approach, and verification.

By contributing, you agree that your contribution is licensed under this project's MIT License and that you have the right to submit it. Please follow the [Code of Conduct](CODE_OF_CONDUCT.md).
