# Changelog

This project follows [Semantic Versioning](https://semver.org/). Dates use ISO 8601.

## [1.1.0] - 2026-07-17

- Preserved multiselections across responsive reflow through semantic text-run restoration.
- Added visual-viewport-aware popup and touch-control positioning and hardened overlay sizing against theme CSS.
- Expanded source documentation, WordPress appearance controls, and minified operational plugin assets.
- Allowed finer WordPress brush tuning with 0.01 increments and a 0.001 Fog fade-speed minimum.
- Added complete WordPress.org compatibility metadata through WordPress 7.0.

## [1.0.0] - 2026-07-13

- First public release of the browser library.
- Added complete public, configuration, and internal-function documentation.
- Added readable and minified browser distributions with a dependency-free build process.
- Added the Clean Selection WordPress plugin.
