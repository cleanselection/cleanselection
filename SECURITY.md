# Security policy

Security fixes are provided for the latest released major version.

Please do not open a public issue for a suspected vulnerability. Use [GitHub private vulnerability reporting](https://github.com/cleanselection/cleanselection/security/advisories/new) and include affected versions, impact, reproduction steps, and any suggested mitigation. You should receive an initial response within seven days.
