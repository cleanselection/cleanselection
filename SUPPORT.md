# Support

Use [GitHub Discussions](https://github.com/cleanselection/cleanselection/discussions) for integration questions and ideas. Use the issue tracker for reproducible bugs. Include the Clean Selection version, browser and operating system, a reduced HTML example, configuration, and exact reproduction steps.

Security reports belong in [private vulnerability reporting](SECURITY.md).
